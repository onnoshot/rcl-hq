## Teknik Stack
- Saf HTML + CSS + Vanilla JS (framework yok)
- Font: SF Pro Display sistem stack (Google Fonts yok)
- Deploy: Vercel (static hosting, vercel.json mevcut)
- Hero: Canvas particle network animasyonu (kırmızı noktalar + bağlantılar, beyaz bg)
- Renk sistemi: Beyaz (#fff) / Siyah (#0a0a0a) / Kırmızı (#cc1c1c)
